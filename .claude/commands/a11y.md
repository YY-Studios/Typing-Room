Audit the current component or file for accessibility issues. Follow these standards:

**Semantic HTML**

- div/span with onClick must be button or a
- Heading hierarchy must be sequential (h1 → h2 → h3)
- Form elements must have associated label

**ARIA**

- Icon-only buttons must have aria-label
- Decorative images: alt="" (empty string)
- Meaningful images: descriptive alt text
- Dynamic content changes: consider aria-live

**Keyboard navigation**

- All interactive elements must be keyboard-accessible
- Custom dropdowns/modals must implement focus trap
- On modal close: return focus to trigger button
- focus-outline-none alone is forbidden — must have custom focus-visible style

**Color contrast**

- Normal text: minimum 4.5:1 ratio
- Large text (18px+ or bold 14px+): minimum 3:1
- Never convey information by color alone

**Touch targets**

- All buttons/links: minimum 44×44px
