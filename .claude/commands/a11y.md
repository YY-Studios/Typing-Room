Audit the current component or file for accessibility issues.

Check for:

1. **Semantic HTML** — div/span with onClick instead of button/a, missing label on form elements, heading hierarchy issues
2. **ARIA** — icon-only buttons missing aria-label, decorative images missing alt="", meaningful images missing descriptive alt text
3. **Keyboard navigation** — interactive elements not reachable by keyboard, missing focus styles (outline-none without replacement)
4. **Color contrast** — text on colored backgrounds that may fail 4.5:1 ratio
5. **Touch targets** — buttons/links smaller than 44×44px

Report findings as a checklist:

- ✅ Pass / ⚠️ Warning / ❌ Fail for each category
- List specific line numbers or element names with issues
- Suggest fixes inline

Do NOT auto-fix — report only.
